Create line art illustrations for a Marathi book on Artificial Intelligence (AI). Each chapter requires one illustration based on the English description of the AI topic.

---

### Key Guidelines

#### 1. Style and Aesthetics
- Line Art Only: Strictly black-and-white line art. Ignore any color references in descriptions.
- Background: Pure white (no shades of gray).
- Human Figures: Replace stick figures with cartoonish or filled human figures (e.g., simple, expressive, and relatable).
- Consistency: Maintain a 4x6-inch size for all illustrations.
- Perspective:
  - Preserve the original nature of the diagram (e.g., 2D chart, scientific diagram, 3D).
  - For 3D illustrations, ensure realistic perspective and depth.

#### 2. Content and Accuracy
- Primary Reference: The illustration description given in English (to ensure all key elements are included).
- Body Text: Use Marathi for all labels/annotations. Simple letters (e.g., "A," "B") can remain in English.
- Clarity: The illustration should instantly convey the mental model to the reader. Avoid clutter; prioritize minimalism and effectiveness.

#### 3. Workflow
1. Understand the AI topic: Study the description and sketch to grasp the core concept.
2. Plan: Outline how to merge the sketch’s style with the description’s details.
3. Clarify Doubts: Ask for input if the sketch or description is ambiguous.
4. Execute: Create the refined line art, ensuring alignment with the sketch and description.

---

### Deliverables
- One illustration per article, adhering to:
  - Style (line art, B&W, 4x6 inches).
  - Language (Marathi text + English title).
  - Clarity (self-explanatory visual).
