## शीर्षकाविषयी

"एआय-रूपे" या शीर्षकात 'रूपे' शब्दाचे तीन अर्थ सूचित होतात.

**एक** म्हणजे 'रूप' चे बहुवचन — एआय (AI, आर्टिफिशिअल इंटेलिजन्स, कृत्रिम बुद्धिमत्ता) ची अनेक रूपे.

**दुसरा** अर्थ त्याचे प्रकट होणे, जसे "— रूपे प्रकट झाले" — विविध क्षेत्रांत प्रकट होणे.

**तिसरा** अर्थ 'रूप' शब्दाची सप्तमी विभक्ती, म्हणजे 'रूपामध्ये'. हे शीर्षक एआयमध्ये काय दडले आहे याचा उहापोह सुद्धा या पुस्तकात आहे हे सुचवत आहे.

## एआय शब्दकोश
*मराठी–इंग्रजी संज्ञा*

| मराठी संज्ञा | इंग्रजी संज्ञा | थोडक्यात अर्थ |
|:---|:---|:---|
| कृत्रिम बुद्धिमत्ता | Artificial Intelligence (AI) | मानवी बुद्धीप्रमाणे काम करणारी संगणक प्रणाली |
| यंत्र बुद्धिमत्ता | Machine Learning (ML) | डेटावरून स्वयं-शिकणारी संगणक प्रणाली |
| बृहत भाषा प्रारूप | Large Language Model (LLM) | अजस्त्र डेटावर प्रशिक्षित भाषा प्रणाली (उदा. GPT) |
| सृजन कृत्रिम बुद्धिमत्ता | Generative AI (Gen-AI) | नवीन मजकूर, चित्र, आवाज निर्माण करणारी एआय |
| कृत्रिम बुद्धिमत्ता कर्मचारी | AI Agent | स्वयंचलितपणे कृती करणारी एआय प्रणाली |
| मज्जातंतू जाळे | Neural Network | मानवी मेंदूपासून प्रेरित संगणकीय रचना |
| सूचक-प्रश्न | Prompt | एआयला दिली जाणारी सूचना किंवा प्रश्न |
| सूचक-प्रश्न अभियांत्रिकी | Prompt Engineering | परिणामकारक प्रश्न/सूचना लिहिण्याचे कौशल्य |
| बृहत माहिती | Big Data | प्रचंड प्रमाणात जमा होणारी डिजिटल माहिती |
| आकृतिबंध | Pattern | डेटामधील पुनरावृत्ती होणारी रचना |
| प्रशिक्षण | Training | डेटा देऊन एआय प्रणाली शिकवणे |
| प्रारूप | Model | डेटावरून तयार झालेली एआयची गणितीय रचना |
| पूर्वग्रह | Bias | डेटा किंवा प्रणालीतील एकांगी झुकाव |
| विशद एआय | Explainable AI (XAI) | निर्णयाचे कारण स्पष्ट करणारी एआय |
| जबाबदार एआय | Responsible AI | नैतिक आणि जबाबदार एआय वापर |
| संरक्षक-झापडे | Guardrails | एआयच्या वर्तनावर घातलेल्या नियंत्रणाच्या मर्यादा |
| बनावटी व्हिडिओ | Deepfake | एआयने तयार केलेले खोटे-सत्य वाटणारे व्हिडिओ |
| डेटा-अद्वैत | Data-Advaita | माणूस आणि त्याचा डिजिटल डेटा यांचे एकत्व |
| लक्ष अर्थव्यवस्था | Attention Economy | मानवी लक्ष हेच उत्पन्नाचे साधन मानणारी अर्थव्यवस्था |
| डिजिटल उपवास | Digital Fasting | ठराविक काळ डिजिटल माध्यमे टाळणे |
| डिजिटल किमान | Digital Minimalism | अनावश्यक डिजिटल गोष्टी कमी करणे |
