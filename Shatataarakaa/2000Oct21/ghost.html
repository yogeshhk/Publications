Ghostview for sharp printouts

To install GhostView, one needs to install "GhostScript" first, even if you will never use it. It will take up some space, but the installation does not take long. Please download and install it for your Operating System (Unix, Windows, Mac) from:

http://www.cs.wisc.edu/~ghost/aladdin/get600.html

Now we can install GhostView, which takes equal - if not lesser time from:

http://www.cs.wisc.edu/~ghost/gsview/get29.html

If you have any difficulties, please let me know. Thank you!
